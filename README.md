# wheretogo
> Where should I go yah?

No longer you'll be confused where to go. `wheretogo` will provide you Facebook events nearby.

# Installation

`$ npm i wheretogo -g`

# Usage

```
$ wheretogo --help

  Usage
    $ wheretogo [text]

  Example
    $ wheretogo 'bandung'
    [eventResults]

  Run it without arguments to enter the live search
```
